<b> Gen AI Reflection </b>


Using Chatgpt has honestly made the coursework way more manageable for me and easier to understand. Instead of trouble shotting for hours or searching for random fixes, being able to ask questions and get immediate feedback has made a huge difference.

I’ve mostly used it when I get stuck or something isn’t working the way I expect. I can describe the issue and usually get pointed in the right direction. It’s helped me understand what’s actually going wrong instead of guessing, which feels more productive. I’ve also used it to help me think through assignments, like how to structure things or where certain pieces of code should go.

Instead of relying on Chatgpt to do all the work for me, I do try to limit the use to one of a tool. As someone who started the Media Arts program long before student use of AI, seeing the contrast has been super interesting in both myself and others. I believe that if used correctly, AI has the potential to open coding to a greater range of people in a creative format. It’s also personally made me more willing to experiment, because I know if I break something I can figure it out instead of getting completely stuck.

Overall, it’s been really helpful for answering questions, troubleshooting, and making the process less time consuming. While complete reliance is something that I do think we need to avoid, using Gen AI as a resource/tool for coursework has been extremely valuable to me. I am looking forward to building on the foundational skills learned in this class for future works. I do also agree that it is important to be able to back up our work with actual knowledge, so while Gen AI is a helpful tool, it cannot replace actually learning code or thinking for ourselves.
